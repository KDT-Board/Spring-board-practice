## 유저 기능
    
회원가입시 아이디와 닉네임은 중복이 불가하다.

아이디, 비밀번호로 로그인을 할 수 있다.

비밀번호와 닉네임 수정이 가능하다

회원 탈퇴시, 탈퇴한 유저가 작성한 글, 댓글, 좋아요 모두 삭제된다.

마이 페이지에서 본인 정보 확인, 수정 및 본인이 작성한 글, 댓글을 추가한 글, 좋아요 누른 글 리스트를 확인할 수 있다.
